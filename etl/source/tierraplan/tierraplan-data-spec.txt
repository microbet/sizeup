== Provision ==

Tierraplan sends a new version of the Demographic/CEX data set to SizeUp every 6 months.

Each data set is associated with an alphanumeric version string (e.g. "2017A"), which includes the effective year of the data (e.g. "2017A" indicates the data is correct as of 2017).

To send it, Tierraplan uploads a set of files to SizeUp's S3 bucket (arn:aws:s3:::tierraplan-transfer, available to IAM user sizeup/TierraPlan). The files can be uploaded individually under the pathnames below, or they can be archived into a zipfile using the pathnames below. If a zipfile is used, the zipfile is uploaded to the bucket at /AGS-${version string}.ZIP .

After successful upload (or failure SizeUp can fix) Tierraplan sends email notification to data@sizeup.com .

Payment terms: Time and materials. (AGS data license is not included.)
